# IT Support Ticket Analysis Dashboard | Power BI

## Overview
This project presents an interactive Power BI dashboard developed to analyze IT support ticket data. The dashboard provides insights into ticket performance, service efficiency, and SLA compliance to support data-driven decision-making.

## Objectives
- Analyze IT support ticket performance.
- Monitor ticket status and priorities.
- Track SLA compliance.
- Measure resolution efficiency.
- Identify monthly support trends.

## Skills Demonstrated
- Data Cleaning (Power Query)
- Data Transformation
- DAX Measures
- Data Visualization
- KPI Development
- Interactive Dashboard Design
- Slicers & Filters
- Trend Analysis

## Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX

## Key Insights
- Monitored ticket volume and support trends.
- Evaluated SLA compliance.
- Analyzed ticket priorities.
- Tracked resolution time.
- Supported operational decision-making through interactive dashboards.

## Dataset
Source: Public dataset from Kaggle.
